## 四、动效规格

| 位置 | 动效类型 | 参数 |
|------|----------|------|
| 全局背景 | 粒子星云漂移（tsParticles） | 80粒子，速度 0.3，连线 opacity 0.2 |
| Hero 标题 | 打字机逐字出现 | 每字 120ms，光标闪烁 |
| Hero 副标题 | fade-in-up | delay 0.8s，duration 0.6s |
| 按钮 | 发光脉冲（box-shadow 呼吸） | 2s 周期 |
| S2 卡片 | 滚动触发浮起 | stagger 0.15s，translateY 40px→0 |
| S3 手机 | 3D 视差倾斜 | 鼠标联动，最大 ±15deg |
| S4 数字 | 滚动计数 | duration 1.5s，ease-out |
| S4 背景 | 流光扫过 | 线性渐变 translateX，3s |
| S5 Tab | 截图滑入 | translateX 60px→0，0.3s ease |
| S6 背景 | conic-gradient 旋转 | 20s 匀速，opacity 0.4 |
| 向下箭头 | 呼吸弹跳 | translateY 0↔8px，1.5s ease-in-out |

## 五、技术方案

| 项目 | 选型 | 原因 |
|------|------|------|
| 技术栈 | 纯 HTML + CSS + JavaScript | 直接部署到 Nginx，零运行时依赖 |
| 动效 | GSAP 3 + ScrollTrigger（CDN） | 行业标准，性能最优，免费可商用 |
| 粒子 | tsParticles（CDN） | 轻量，配置灵活 |
| 图标 | Lucide Icons（CDN） | 现代风格 SVG |
| 字体 | Google Fonts（Noto Sans SC + Inter） | 思源黑体中文效果最佳 |

> **单文件方案**：所有 CSS/JS 内联到一个 `index.html`，SCP 上传即生效。

## 六、部署方案

```nginx
# 现有 Nginx 配置补充

    root /var/www/static;
    index index.html;
}

location /privacy-policy {
    alias /var/www/static/privacy_policy.html;
    default_type text/html;
}
```
