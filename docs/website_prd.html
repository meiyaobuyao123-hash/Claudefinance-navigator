5技术方案

项目选型原因
技术栈纯 HTML + CSS + JavaScript直接部署到 Nginx，零运行时依赖
动效GSAP 3 + ScrollTrigger（CDN）行业标准，性能最优，免费可商用
粒子tsParticles（CDN）轻量，配置灵活
图标Lucide Icons（CDN）现代风格 SVG
字体Google Fonts（Noto Sans SC + Inter）思源黑体中文效果最佳
⚡ 单文件方案：所有 CSS/JS 内联到一个 index.html，零配置部署，SCP 上传即生效。

6部署方案

# Nginx 配置（在现有 sites-enabled 文件中已有基础配置）
server {
    server_name www.ai100trading.cn ai100trading.cn;

    location / {
        root /var/www/static;
        index index.html;
    }

    location /privacy-policy {
        alias /var/www/static/privacy_policy.html;
        default_type text/html;
    }
}

# 部署命令
scp index.html root@43.156.207.26:/var/www/static/index.html

7验收标准

  • Lighthouse 性能分数 ≥ 80
  • 移动端完美适配（375px 最小宽度）
  • 所有动效在 iOS Safari 正常运行
  • 首屏加载 < 3s（中国大陆网络）
  • App Store 下载链接可点击跳转
  • 隐私政策链接 /privacy-policy 正常访问
  • 粒子背景不影响滚动性能（60fps）
  • 页面无水平滚动条
