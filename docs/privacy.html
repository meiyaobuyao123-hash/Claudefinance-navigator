隐私政策

最后更新：2026年3月6日

欢迎使用「来理财啊」（以下简称"本应用"）。本应用由个人开发者运营，我们非常重视您的隐私保护。本政策说明我们如何收集、使用和保护您的信息。

一、我们收集的信息

  • 账户信息：当您注册或登录时，我们收集您的电子邮件地址，用于账户认证。
  • 设备标识符：我们生成一个随机的匿名设备 ID，用于在不同设备间同步您的持仓和配置数据。此 ID 与您的真实身份无关。
  • 用户输入数据：您主动录入的基金代码、持仓份额、成本价格、资产配置比例等财务信息，仅用于向您展示收益计算结果。
  • 使用行为数据：我们不收集任何崩溃以外的行为分析数据。

二、我们如何使用信息

  • 提供核心功能：实时行情查询、收益计算、资产配置评估、AI 理财问答。
  • 云端数据同步：通过 Supabase 安全存储您的持仓记录，支持多设备访问。
  • 本地推送通知：止盈止损提醒、每日收益推送（仅在您设备本地触发，不上传通知内容）。

三、信息共享

我们不会出售、交易或以其他方式向第三方转让您的个人信息，以下情况除外：

  • Supabase：用于云端数据存储，数据经加密传输，详见 Supabase 隐私政策。
  • Anthropic Claude API：AI 对话功能会将您的问题文本发送至 Claude API，不包含任何个人身份信息，详见 Anthropic 隐私政策。
  • 行情数据接口：查询基金和股票行情时会访问天天基金、新浪财经、Yahoo Finance 的公开接口，仅传递证券代码，不包含用户信息。

四、数据安全

我们采用行业标准的安全措施保护您的数据，包括 HTTPS 加密传输和行级安全策略（Row Level Security）。您的财务数据仅与您的匿名设备 ID 关联，我们无法将其与您的真实身份对应。

五、数据留存与删除

您可以随时在"我的"页面注销账户。注销后，我们将删除与您账户关联的所有数据。如需删除本地设备数据，卸载本应用即可。

六、未成年人保护

本应用面向18岁及以上成年用户。我们不会故意收集未成年人的个人信息。

七、政策更新

如本政策发生重大变更，我们将在应用内通知您。继续使用本应用视为接受更新后的政策。
